# HomeBase Budget

Real-time household budget tracker for two. Built with React + Vite + Firebase Firestore.

## First-time setup (one time only)

```bash
npm install
```

## Run locally

```bash
npm run dev
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

Live at: https://rsandlin22.github.io/homebase-budget/
